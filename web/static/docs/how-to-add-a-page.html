Home > Page Management > How to add a page

Summary

In the "Pages" tab you can add pages. If you add a page, it will be visible for the users of your website.

Example

To add a page of a certain type,

  1. open NodeCMS admin panel
  2. Click on the "Pages" tab
  3. Click on the "add page" button of the certain type
