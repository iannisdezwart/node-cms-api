Summary

The "Files" tab is the place where you can manage the files of your website. These files could for example be images or videos you would like to display on your website. After you added some files, you can add them to your pages later by adding or editing a page.

To open the file manager,

  1. open NodeCMS admin panel
  2. Click on the "Files" tab as visualised below
  3. You might be promted to enter your password. In this case, fill in your password.

Then, you will see a table like this:

1 (red): The table containing the files and folders in the current folder you are in

2 (dark blue): Some information about the file size and last modified date of a file

3 (green): Actions you can perform on a file or a folder (see this page for more help)

4 (yellow): Actions you can perform on the current folder you are in (see this page for more help)

5 (light blue): The path to the folder you are currently in
