Home > Page Management > How to edit a page

Summary

In the "Pages" tab you can edit pages. If you edit a page, it will automatically be updated for the users of your website.

Example

To edit a certain page,

  1. open NodeCMS admin panel
  2. Click on the "Pages" tab
  3. Click on the "edit" button of the certain page

See the example below:

Clicking on "edit page" in the above example will bring you to the page editor. You will see something like this:

The bold items outlined in red are fields.
Each field has an input right below it. The inputs look different, depending on their type. The inputs will be filled in with their current data.
In this example, the field "title" has a simple text input with the text "my-first-page", "photo" has an image input with an image of a fox and "text" has a Word-like text input with the text "Hello, this is my first text page!".
There are seperate guides on how to operate these inputs.
