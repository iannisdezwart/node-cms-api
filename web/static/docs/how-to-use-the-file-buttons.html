Summary

There are four file/folder action buttons, which can be found on the right side of each file/folder in the file table (see the image above). If you use a small screen, you might have to scroll the table to the right to see the buttons.

Note that you will not see any buttons if there are no files or folders in the folder you currently are in.

What each of the buttons does is described thoroughly below:

Detailed description of the file buttons

"Copy" button

The "Copy" button is used for copying a file (or a folder) from one folder to another. It does the same as the "Move" button, except copying keeps the original file, whereas moving removes the original.

When you click on the "Copy" button, you will see a popup like this:

First select a folder by clicking on it. In this example I will copy a file to the "My Folder" folder.
(Optionally, you could change the file name of the copied file below "Fill in the name of the file".)
Then click on "Select folder". Now the file will get copied and you will get a confirmation popup message like this:

"Move" button

The "Move" button is used for moving a file (or a folder) from one folder to another. It does the same as the "Copy" button, except moving removes the original file, whereas copying keeps the original.

When you click on the "Move" button, you will see a popup like this:

First select a folder by clicking on it. In this example I will move a file to the "My Folder" folder.
(Optionally, you could change the file name of the moved file below "Fill in the name of the file".)
Then click on "Select folder". Now the file will get moved and you will get a confirmation popup message like this:

Be careful not to move any files which are currently used on pages of your website, as the affected pages will not load their content correctly if done insecurely.
If you really want to move a file that is used on your pages,

  1. Copy the file to the new desired location
  2. Change all references to the file by editing all pages that have references in the page manager
  3. Delete the original file

"Rename" button

The "Rename" button is used to rename a file or folder.

When you click on the "Rename" button, you will see a popup like this:

First select a folder by clicking on it. In this example I will rename the file "video.mp4" to "drone-footage.mp4".
Enter the new name and then click on "Rename". Now the file will get renamed and you will get a confirmation popup message like this:

Be careful not to rename a folder which contains files that are currently used on the pages of your website, as the affected pages will not load their content correctly if done insecurely.
If you really want to rename a file that is used on one of your pages,

  1. Copy the file to the same folder, but under a different name
  2. Change all references to the file by editing all pages that have references in the page manager
  3. Delete the original file

"Delete" button

The "Delete" button is used to permanently delete a file or folder (including all its content).
Warning: this action cannot be undone.

When you click on the "Delete" button, you will see a popup like this:

Click on the "Delete" button of this popup if you are sure you want to delete the file. You will not get a confirmation popup message.

Be careful not to delete a file or a folder which contains files that are currently used on the pages of your website, as the affected pages will not load the deleted content anymore.
