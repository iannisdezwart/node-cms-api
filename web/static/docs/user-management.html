Summary

The "Users" tab is the place where you can manage the users who have access to the NodeCMS admin panel.

To open the user management tab,

  1. open NodeCMS admin panel
  2. Click on the "Users" tab as visualised below
  3. You might be promted to enter your password. In this case, fill in your password.

Then, you will see a table like this:

1 (red): The user management view

2 (green): The names of the users

3 (dark blue): Actions you can perform on users (see below)

4 (yellow): "Add User" button

Action buttons

"Change Password" button

To change the password of a certain user, click the "Change Password" button of that user. You will see a popup like this:

Fill in the current password in the first input, and the desired new password in the second and third. Then click on the "Enter" button.

The user's password is now changed and you will get a confirmation popup message like this:

"Delete" button

To delete a certain user, click the "Delete" button of that user. You will see a popup like this:

Click on "Delete User" if you are sure you want to delete this user. The user will then be deleted. You will not receive a confirmation popup message.

"Add User" button

To add a user, click the "Add User" button. You will see a popup like this:

Fill in the name of the user in the first input, and the desired new password in the second and third. Then click on the "Enter" button.

The user is now added. You will not receive a confirmation popup message.
