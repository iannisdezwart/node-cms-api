Summary

There are three folder action buttons, which can be found at the top of the file table (see the image above).

What each of the buttons does is described thoroughly below:

Detailed description of the file buttons

"Up a level" button

The "Up a level" button will bring you to the parent folder (the folder that contains the folder you are currently). For example: if you are in the folder "/my-folder/sub-folder", this button will bring you to "/my-folder".

"Upload Files" button

The "Upload Files" button can be used to upload files to the folder you are currently in.
Click the button, this will open a system menu where you can select files from your computer.

"New Folder" button

The "New Folder" button will create a new (sub) folder in the folder you are currently in.
Click the button, enter the desired name and click "Create".
