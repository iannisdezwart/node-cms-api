Summary

You will encounter different input types when you add or edit a page.

There are five different inputs:

Simple text

The simple text is the easiest to operate. Simply fill in the desired text.



Word-like text

This input is a more advanced text editor, like Word. It supports text formatting.



Date

This input is used for dates. It might look different on your browser. If you click on the the input, or on the icon at the edge of it, you will see a calendar where you can select a date.



Images

With the images input you can add images to a page.



You might not see the rectangle with a plus. This means that you can only add one image to this input.

If you hover your cursor on a photo, you might see some buttons and arrows. You can use the arrows to change the order of the images.

If you click on the delete button of a certain image, this image will be removed from the image list.

If you click on "Edit" or on the rectangle with the plus, you will get a popup that looks like this:



First click the image you want to add, then click on "select". This will add or update the selected image.

Note that you have to first add images in the file manager before they will appear in the above menu.

Video

With the video input you can add a video to a page.



To change the video, click on the "Edit" button. You will see a popup that looks like this:



First click the video you want to add, then click on "select". This will update the video.

Note that you have to first add videos in the file manager before they will appear in the above menu.
